# Calculadora Simples em Python utilizando Tkinter

Este projeto é uma calculadora simples com interface gráfica, desenvolvida em Python utilizando a biblioteca **Tkinter**.  
O objetivo principal é praticar conceitos de programação, interfaces gráficas, organização de código e uso do Git/GitHub.

---

## Funcionalidades

- Interface gráfica inspirada em calculadoras de celular
- Modo **claro / escuro**
- Operações básicas:
  - Adição (+)
  - Subtração (-)
  - Multiplicação (*)
  - Divisão (/)
- Botão de apagar último caractere (⌫)
- Botão de limpar (C)
- Botão de igual ocupando duas linhas
- Animação simples ao clicar nos botões
- Suporte ao teclado

---

## Tecnologias utilizadas

- **Python 3**
- **Tkinter** (biblioteca padrão do Python para interfaces gráficas)

---

## como utilizar
1. Certifique-se de ter o **Python 3** instalado
2. Clone este repositório ou baixe os arquivos
3. Navegue até a pasta do projeto e execute: 
  python calculadora.py
