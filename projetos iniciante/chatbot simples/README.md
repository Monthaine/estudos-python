# Chatbot Simples em Python 
O projeto foi desenvolvido à partir do projeto de criação de um chatbot simples pelo **Hashtag Programação**.

Trata-se de um projeto **didático**, com o objetivo de praticar conceitos básicos da linguagem Python.

## objetivos do projeto:
- lógica de programação
- uso de loops
- listas e dicionários
- consumo de uma API externa (OpenAI)
- O chatbot mantém o histórico da conversa para responder de forma contextual.

## Como executar o projeto
1. Clone o repositório
```bash
git clone https://github.com/Monthaine/estudos-python.git

2. Acesse a pasta do Bot

3. Instale todas as dependencias necessárias
python -m pip install openai

4. configure uma variável
OPENAI_API_KEY com sua chave openai
ex: setx OPENAI_API_KEY "sua_api_key_aqui"

5. execute o chatbot

obs1. chave API não incluída por motivos de segurança
obs2. o projeto é de fins didáticos e faz parte do processo de aprendizado python.
